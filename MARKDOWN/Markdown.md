#Markdown操作指南
##作用
标记文本的样式
##好处

- 兼顾了「什么人都能打开」和「样式」
- Markdown 转 HTML 非常方便。
##在VSCode中使用
安装插件：Markdown Preview Enhanced
增加并编写文件：***.md
在文件内容上右键，选择Open Preview to slider，右侧将呈现Markdown的样式。
